# 🔥 FIRE Planner - AI Financial Money Mentor

AI-powered personal finance mentor that generates a complete FIRE (Financial Independence, Retire Early) roadmap with SIP planning, asset allocation, tax optimization, and monthly investment guidance.

## 🚀 Overview
The FIRE Planner combines deterministic financial modeling with generative AI (Google Gemini) to provide personalized, actionable financial roadmaps. Specifically tailored for the Indian financial context (INR), it accounts for inflation, asset allocation, and goal-based SIP requirements.

## 🧠 Features
- **FIRE Corpus Calculator**: 25x rule adjusted for future inflation.
- **Goal-Based Planning**: Calculate precise monthly SIPs for specific life goals (House, Education, etc.).
- **Smart Asset Allocation**: Dynamic Equity/Debt/Gold split based on age and custom risk appetite.
- **AI Financial Mentor**: Personalized insights and warnings generated by Google Gemini AI.
- **Interactive Dashboard**: Modern glassmorphism UI with Chart.js visualizations.

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React (Vite, Tailwind CSS v4)
- **AI**: Google Gemini API (gemini-2.5-flash)
- **Data Handling**: Pydantic for validation
- **Tooling**: Axios, Chart.js, Lucide Icons

## ⚙️ Setup & Installation

### 1. Prerequisites
- Python 3.10+
- Node.js 18+
- Google Gemini API Key ([Get one here](https://aistudio.google.com/))

### 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure Environment Variables:
   Create a `.env` file in the `backend/` folder:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```
4. Start the server:
   ```bash
   python app.py
   ```
   *The API will be available at http://127.0.0.1:8000*

### 3. Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   *The app will be available at http://localhost:5173*

## 📈 Methodology
- **Inflation**: Defaulted at 6% (standard for India).
- **Asset Allocation**: Uses the "100 - age" rule, adjusted by +/- 10% based on risk appetite.
- **Expected Returns**: Blended portfolio returns ranging from 8% (Low) to 15% (High).
- **FIRE Number**: Calculated as 25x of annual expenses at the time of retirement (inflation-adjusted).
